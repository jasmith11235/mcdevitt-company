# The McDevitt Company

Commercial real estate advisory firm website. Built with Next.js 14, Tailwind CSS, and Keystatic CMS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) for the site.  
Open [http://localhost:3000/keystatic](http://localhost:3000/keystatic) for the content editor.

## Content Editing

This site uses [Keystatic](https://keystatic.com) for content management. Non-technical editors can update all site content through the visual editor at `/keystatic`.

Editable content includes:
- Testimonials
- Office locations
- Select projects
- News/Reading Room articles
- Page copy (hero, about, services)

## Deployment

Deploy to Vercel:

```bash
npx vercel
```

Or connect the GitHub repo to Vercel for automatic deployments on push.
